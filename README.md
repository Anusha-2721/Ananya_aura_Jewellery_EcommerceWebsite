# 💎 Ananya Aura Jewellery

## 📌 Project Overview

Ananya Aura Jewellery is a full-stack e-commerce web application developed for an online jewellery shopping platform.

The main aim of this project is to provide users with a smooth and user-friendly jewellery shopping experience where users can create an account, login, explore products, search jewellery items, add products to cart, place orders, and manage their profile.

This project demonstrates the integration of frontend technologies, backend APIs, and database management to build a complete full-stack application.

---

# 🎯 Objective

The objective of this project is to develop a complete jewellery shopping application by implementing:

- User authentication system
- Product browsing and filtering
- Cart management
- Order processing
- Database integration
- Backend API communication

The project focuses on creating a responsive, organized, and scalable web application.

# ✨ Features Implemented

## 👤 User Module

- User Registration
- User Login Authentication
- User Profile Management
- Logout Functionality


## 💍 Product Module

- Display jewellery products
- Product images and pricing
- Search jewellery items
- Category filtering (Men/Women)


## 🛒 Cart Module

- Add products to cart
- View cart items

- Calculate total amount


## 📦 Order Module

- Customer checkout page

---

# 🗄️ Database Tables


## users table

Stores user account information:

- User ID
- Username
- Phone
- Date of Birth
- Email
- Password


## products table

Stores jewellery product information:

- Product ID
- Product Name
- Price
- Image
- Category


## cart table

Stores cart items:

- Cart ID
- User ID
- Product Name
- Price
- Image
- Quantity


## orders table

Stores order information:

- Order ID
- User ID
- Product Name
- Price
- Quantity
- Total Amount
- Order Status
- Order Date
